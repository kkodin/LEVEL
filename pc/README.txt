曲線推進管理システム 分割HTML版

使い方:
1. 00_基本情報入力_保存読込.html を開き、現場・土質・管・線形を入力します。
2. 「保存JSONを作成」で JSON を保存します。
3. 各個別HTML（01〜04）を開き、同じJSONを読み込むか、00から各ボタンで開きます。

目的:
入力・線形計算・ビジュアル・目地開口・推力管理を分離し、複雑さと不具合の影響範囲を小さくします。

v14追加:
